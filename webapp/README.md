# Development
```
pnpm install
pnpm run dev
pnpm run build
```

# Package
```
# If you want the web to start with the project, you can execute 
./mvnw clean install

# If you want to run the web project alone, you can execute 
cd ./webapp
pnpm install
pnpm run build
```
